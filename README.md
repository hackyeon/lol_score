## LOL 전적검색 앱
### 개발 목적
riot games open api를 이용한 전적 검색앱 제작
### 개발 환경</h3>
window10<br>
ansroid studio 4.2.0<br>
kotlin 1.5.20<br>
min api level 23

### 라이브러리
통신: Retrofit, OkHttp<br>
데이터베이스: Room<br>
이미지: Glide<br>
